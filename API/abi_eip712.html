function anchorMerkleRoot (bytes32 merkleRoot, bytes32 batchId, bytes quorumAttestation, uint256 committedAt)
Anchors a Merkle root on-chain. Requires HybridShield custodian quorum. Emits MerkleRootAnchored. Foundation for NL=NA Layer 5 enforcement: only logHashes provably included in an anchored root can produce valid Permission Tokens.
ParameterTypeNote
merkleRootbytes32SHA-256 root of TGLF batch
batchIdbytes32Unique batch identifier
quorumAttestationbytesABI-encoded CustodianQuorumAttestation array. operationType: 0 (ANCHOR_MERKLE)
committedAtuint256Unix epoch timestamp of batch commit
reverts QuorumNotMet
function registerPermissionToken (bytes32 tokenId, bytes32 logHash, bytes32 laneOriginHash, bytes32 merkleRoot, ...)
NL=NA Layer 5 enforcement. Registers a Permission Token on-chain. Reverts NLNAViolation if logHash is not provably included in an anchored Merkle root via verifyMerkleInclusion, or if laneOriginHash does not equal keccak256("GOVERNANCE_LANE"). Terminal enforcement gate: no actuation may proceed without a registered, valid Permission Token whose logHash is on-chain verified.
ParameterTypeNote
logHashbytes32Layer 5 — must be in anchored Merkle root
laneOriginHashbytes32Layer 2 — must equal keccak256("GOVERNANCE_LANE")
merkleRootbytes32Previously anchored root containing logHash
merkleProofbytes32[]Ordered sibling hash array forming inclusion proof path
expiresAtuint256Unix epoch expiration. Actuation MUST reject if block.timestamp ≥ expiresAt
signerKeyIdbytes32HSM key identifier for Governance Lane signature verification
signatureValuebytesBase64url-decoded HSM signature over PermissionToken fields
reverts NLNAViolation(violationReason) 0=MERKLE_NOT_FOUND, 1=INVALID_LANE_ORIGIN
view verifyPermissionToken (bytes32 tokenId, uint256 currentTimestamp) → (bool valid, uint8 revocationStatus, int256 remainingLifetimeMs)
Read-only verification of Permission Token validity, revocation status, and remaining lifetime. Actuation layer calls this before firing. revocationStatus: 0=ACTIVE, 1=REVOKED_BY_EMERGENCY, 2=REVOKED_BY_TRI_CAMERAL.
pure verifyMerkleInclusion (bytes32 leaf, bytes32 merkleRoot, bytes32[] proof) → bool
Pure function. Verifies that a leaf hash is included in a Merkle root given a proof path. Used internally by registerPermissionToken (NL=NA Layer 5) and exposed for external governance verification.
function activateEpistemicHoldSystemWide (bytes32 holdRationale, bytes32 escrowRecordId, bytes quorumAttestation, uint256 resolutionDeadline)
Activates system-wide Epistemic Hold (State 0). Requires HybridShield custodian quorum. All subsequent Permission Token registrations are blocked until hold is resolved. Emits EpistemicHoldActivated.
reverts QuorumNotMet
function resolveEpistemicHoldSystemWide (bytes32 escrowRecordId, uint8 resolvedState, bytes32 resolutionRationaleHash, bytes quorumAttestation)
Resolves system-wide Epistemic Hold. resolvedState uint8: 1=PROCEED, 0=REFUSE (function-local encoding). State 0 (Epistemic Hold) is constitutionally invalid as a resolution target. Reverts InvalidResolutionState for any other value.
resolvedState valueMeaning
1PROCEED (+1 integer)
0REFUSE (−1 integer). Note: uint8(0) encodes Refuse here, not Epistemic Hold.
any otherreverts InvalidResolutionState
function executeEmergencyOverride (bytes32 overrideRequestId, uint8 overrideType, bytes32 targetDecisionId, uint8 forcedState, ...)
Break-glass override. ALL invocations logged on-chain before execution — governance trail exists before state change fires. NL=NA applies without exception. forcedState uint8: 0=EpistemicHold, 255=Refuse. Proceed (+1) is constitutionally blocked and will revert UnauthorizedOverride. Emits EmergencyOverrideExecuted.
forcedState valueMeaning
0EpistemicHold (State 0)
255Refuse (State −1)
any otherreverts UnauthorizedOverride — Proceed constitutionally blocked
function revokePermissionToken (bytes32 tokenId, uint8 revocationReason, bytes32 revocationMerkleRoot, bytes quorumAttestation)
Revokes a registered Permission Token. Revocation state machine: ACTIVE → REVOKED_BY_EMERGENCY or REVOKED_BY_TRI_CAMERAL. revocationMerkleRoot anchors the revocation on-chain.
revocationReasonMeaning
1REVOKED_BY_EMERGENCY
2REVOKED_BY_TRI_CAMERAL

Emitted when a Merkle root batch is anchored on-chain. anchorId is the unique on-chain identifier for this anchoring event.

Emitted when a Permission Token is successfully registered. logHash and merkleRoot are the NL=NA Layer 4/5 binding fields.
eventEpistemicHoldActivated(bytes32 escrowRecordId, bytes32 holdRationale, uint256 activatedAt, uint256 resolutionDeadline, bytes32 traceId)
Emitted on system-wide Epistemic Hold activation. All downstream actuation gates must observe this event and halt execution.
eventEpistemicHoldResolved(bytes32 escrowRecordId, uint8 resolvedState, bytes32 resolutionRationaleHash, uint256 resolvedAt, bytes32 traceId)
Emitted on resolution. resolvedState: 1=PROCEED, 0=REFUSE (function-local encoding). Never emitted with Epistemic Hold as resolved state — constitutionally prohibited.
eventEmergencyOverrideExecuted(bytes32 overrideRequestId, uint8 overrideType, uint8 forcedState, bytes32 justificationHash, uint256 executedAt, uint256 forcedStateExpiresAt)
Emitted BEFORE state change — governance log precedes execution. This ordering is constitutional: the log exists before the action fires. forcedState: 0=EpistemicHold, 255=Refuse.
eventLanternStatusBroadcast(bytes32 lanternSignalId, bytes32 artifactNameHash, uint8 compliancePosture, uint256 broadcastAt, bytes32 signatureBlockHash)
Goukassian Principle Lantern public beacon. artifactNameHash = keccak256("lantern"). compliancePosture: 0=FULLY_COMPLIANT, 1=EPISTEMIC_HOLD_ACTIVE, 2=REGULATORY_REVIEW, 3=PILLAR_DEGRADED, 4=EMERGENCY_OVERRIDE_ACTIVE.
Custom Errors
errorNLNAViolation(bytes32 tokenId, bytes32 logHash, uint8 violationReason)
violationReason: 0=MERKLE_NOT_FOUND (logHash not in anchored root), 1=INVALID_LANE_ORIGIN (laneOriginHash ≠ keccak256("GOVERNANCE_LANE")). Terminal NL=NA Layer 5 enforcement.
errorEpistemicHoldActive(bytes32 escrowRecordId, uint256 resolutionDeadline)
Reverted when a state-mutating operation is attempted while system-wide Epistemic Hold is active.
errorQuorumNotMet(uint8 provided, uint8 required)
HybridShield custodian quorum insufficient for the requested operation type.
errorInvalidResolutionState(uint8 attemptedState)
Reverted when resolveEpistemicHoldSystemWide is called with a value other than 0 (REFUSE) or 1 (PROCEED). Epistemic Hold re-resolution is constitutionally prohibited.
errorUnauthorizedOverride(uint8 attemptedForcedState, address callerAddress)
Reverted when executeEmergencyOverride is called with forcedState targeting Proceed (+1). Constitutionally blocked. Forced Proceed has no valid uint8 encoding in emergency override context.
errorTokenExpired(bytes32 tokenId, uint256 expiresAt, uint256 currentTimestamp)
Reverted when an actuation attempt references a Permission Token past its expiresAt timestamp. No grace period.

function verifyEconomicRightsCompliance (bytes32 decisionId, tuple baselIiiVector, tuple fatfVector, tuple ioscoVector, bytes32 traceId)
Verifies Basel III + FATF + IOSCO compliance vectors. Returns compliant bool, violationCode bytes32, and forcedState uint8 (0=EPISTEMIC_HOLD, 255=REFUSE). Emits EconomicRightsMandateViolationDetected on non-compliance.
VectorElements
baselIiiVector[lcr] scaled 1e18 (≥1e18=compliant), [nsfr] scaled 1e18, [capitalRatio], [counterpartyExposureWithinLimits], [stressTestPassed]
fatfVector[sanctionsScreened], [sanctionsHit], [pepInvolved], [transactionMonitoringAlert], [sarRequired]
ioscoVector[layeringDetected], [spoofingDetected], [washTradingDetected], [crossMarketManipulationDetected]
function verifySustainableCapitalCompliance (bytes32 decisionId, tuple parisAgreementVector, bytes32 traceId)
Verifies Paris Agreement and ESG compliance. parisAgreementVector: [carbonFootprintVerified, greenBondEligibility, esgScore (scaled 1e18, range 0–100e18), climateAlignedCapitalFlowValidated]. Emits SustainableCapitalMandateViolationDetected on non-compliance.
function verifyGoukassianLicense (bytes32 decisionId, bytes32 licenseToken, bytes32 requestingEntityId, bytes32[] proposedScopeHashes, bytes32[] authorizedScopeHashes)
Verifies Goukassian Principle license scope. violatedArtifact ordinals: 0=none, 1=lantern, 2=signature, 3=license. Exceeded scope triggers automatic Refuse (State −1). Emits GoukassianLicenseViolationDetected on violation.
function recordMandateViolation (bytes32 violationId, bytes32 mandateViolated, uint8 pillarImplicated, bytes32 violationDetailsHash, bytes32 decisionId, uint256 recordedAt)
Records a mandate violation immutably on-chain. Immutable: cannot be modified or deleted once recorded. mandateViolated: keccak256("No Spy"), keccak256("No Weapon"), or keccak256("No Switch Off"). These three mandates are beyond all governance bodies' authority to suspend.
